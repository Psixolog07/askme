# Установка
Для работы вам потребуется `ruby "3.0.4"`, `rails 7`, `gem bundler`

Склонируйте проект и выполните следующие комманды

```
bundle
rails db:migrate
```

# Запуск

Для запуска на локальном сервере выполните команду `rails s`
